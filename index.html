Een korte disclaimer

Deze website heb ik eerder ontworpen en ontwikkeld, ik gebruik dit design nu voor mijn onepager opdracht van het keuzevak HTML/CSS. De website heb ik dus wel zelf geprogrammeerd maar ik heb geen schets meer van het toemalige ontwerp. De CSS komt grotendeels wel uit een libary.

Reflectie
Reflectie keuzevak

In dit keuzevak heb ik vooral geleerd dat ik met HTML-5 tags moet werken in plaats van met HTML-4 tags, ik gebruikte vooral de HTML-4 tags omdat mij dat zo is aangeleerd, ik vind het leuk om in mijn vrije tijd mee te helpen met het ontwikkelen van open-source projecten en het helpen van mensen op websites als Stackoverflow. Ik gebruik daarbij over het algemeen weinig HTML/CSS. Het was goed voor mij om alles weer een keertje door te nemen vanaf het begin. Door regelmatig feedback te vragen heb ik ook geleerd dat er tegenwoordig grids gemaakt kunnen worden met CSS.

Actiepunten
Welke kwaliteiten wil ik ontwikkelen?

Omdat ik vaak CSS gebruik uit andere libary's vergeet ik vaak een aantal tags die ik soms wel nodig heb, zo haal ik position en display vaak door elkaar, door vaker een eigen stylesheet te maken wil ik meer automatisme krijgen in het werken met CSS tags. In plaats van ze even snel te googlen.
